# RStudio Workshop Materials

I created this repository to assist in the understanding and reasoning why we learn R as ecologists.
One question I got from the undergraduates I help mentor was "why R? What's the point?"

So, here is the answer to that question.

## Why do ecologists use R?

Short answer: **R helps ecologists turn raw data into scientific conclusions**

Best Answer:

### 🍂Ecology is Data Driven

-   Modern ecology relies on complex datasets, combining advanced GPS, telemetry, environmental time-series, and observations across space and time.

-   Usually these datasets are often large and messy, and R is specifically designed to handle this type of data efficiently.

### 📊 RStudio Allows for Real Statistical Analysis

-   RStudio supports a wide range of statistical approaches, from basic models to advanced techniques.

### 🎨 RStudio Produces Quality Figures

-   In RStudio you are able to create publishable and shareable figures which can be reproduced later.
    Furthermore, it allows for customization and consistency

### 🌊RStudio is a Valuable Career Skill

-   Most jobs require you know some level of R to be a competitive applicant.

-   Jobs include

    -   Ecology and wildlife biology

    -   Fisheries science

    -   Conservation

    -   Government agencies

    -   Academic research

### 💡In Conclusion...

**R is a programming language and tool for doing better science**
